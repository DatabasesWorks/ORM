Detailed Description

The ORMObject is a base class for all your models.

For create model, you must inherits by this class. Example:

Car : public ORMObject
{
Car();
ORM_PROPERTY(QString, brand)
ORM_PROPERTY(QDate, year)
ORM_PROPERTY(QString, color)
}
