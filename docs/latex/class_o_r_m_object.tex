\subsection{Detailed Description}
The \hyperlink{class_o_r_m_object}{O\-R\-M\-Object} is a base class for all your models. 

For create model, you must inherits by this class. Example\-: 
\begin{DoxyCode}

\}
\end{DoxyCode}
